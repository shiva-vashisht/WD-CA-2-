Student Portfolio Website
A multi‑page personal portfolio website for a BTech CSE student, built using only HTML and basic CSS. It showcases information about me, my skills & projects, and provides a contact form for communication.

Project Overview
This website is part of an academic assignment to design and develop a multi‑page student portfolio using proper HTML structure, semantic layout, and basic CSS. It is fully static and does not use any JavaScript frameworks.

Features
Common navigation bar on all pages

Modern home page with hero section and highlights

About page with enhanced profile, journey timeline, skills overview, and personal story

Skills & Projects page with animated skill cards, project showcase, and achievement stats

Contact page with styled form including Name, Email, Subject, and Message fields

Pages
index.html – Home (hero section, highlights, current work)

about.html – About Me (profile, journey timeline, technical proficiency, personal story)

skills.html – Skills & Projects (skills cards using Flexbox, featured projects, stats)

contact.html – Contact Form (fully styled form + contact information)

style.css – Shared styling for layout, typography, navigation, and components

Technologies Used
HTML5 (semantic tags, structure)

CSS3 (basic styling, gradients, Flexbox, responsive layout)

How to Run
Download or clone this repository.

Open index.html in any modern web browser.

Navigate to other pages using the navigation bar.

No additional setup or build steps are required.
